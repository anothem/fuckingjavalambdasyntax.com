How Do I Declare A Lambda in Java?

This page is still being built, happy to take bug reports and contributions :)

As a method parameter:

@FunctionalInterface
public interface SomeFunction {
public void doWork();
}
void someMethodThatTakesALambda(delegateType lambdaName) {...}

As an argument to a method call:

someObject.someMethodThatTakesALambda ((ParameterClass p) -> {...});

This site doesn't cover every possible use of lambdas.
